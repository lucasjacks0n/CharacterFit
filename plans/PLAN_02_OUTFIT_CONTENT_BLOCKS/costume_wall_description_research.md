Here is costumewall's description seo structure.

## Paragraph 1 — Character Bio + Significance
- Real name + character name  
- Why they’re notable or iconic  
- Persona / vibe description  
- 1–2 achievements or factual highlights  
- Transition sentence to the costume guide  

## Paragraph 2 — Costume Item Summary
- Hype/action phrase  
- One flowing sentence that lists all main costume items
